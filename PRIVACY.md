# 公开范围与隐私处理

这个仓库面向招聘者和技术交流，只保留能够证明项目能力且适合公开的材料。

## 已排除

- 发票、报销单、机票和行程记录。
- 奖金回执、签名文件、承诺书和带个人联系方式的材料。
- 完整 SECOND/UAV-CD 数据集、训练权重和模型缓存。
- 第三方论文全文、重复的历史归档和无关竞赛材料。
- 包含手机号的原始文件名。

## 已保留

- 两份正式获奖证书及证书 PDF。
- 项目文档、决赛答辩 PDF 和系统报告样例。
- 最终集成版核心源码和上游许可证。
- 从最终答辩材料导出的项目结构图、效果图和应用图。

发布到 GitHub 前，仍应检查图片中的二维码、截图中的 API Key 和 PDF 元数据。仓库当前未包含 API Key；LLM 使用说明只要求用户在本地运行时自行填写。
